# EduPlataform Base

from laravel inertia react starter kit


## Config

Configura las variables en setup.json
```json
name : str 
icon : png/jpg/svg
```

## RUN

- clonar el repo
`composer install`
`npm run build`
`composer run dev`

## Deploy

`composer install --optimize-autoloader --no-dev`
`npm run build`

Exporta la base de datos

Ejecuta con php (apache nginx) el archivo public/index.php
